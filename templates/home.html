What is TeamTasker?

TeamTasker is a platform for easy and powerful project management.

Goals of the project.

  • Create sign-in and sign-up.
  • Implement possibility to create projects.
  • Display projects to user.
  • Store tasks for each project.
  • Correctly display tasks for user and project.
  • Allow user to create tasks.
  • Create role system which will be unique for each project.
  • Change available options and displayed content based on role.
  • Give user a possibility to customize role system.
  • Allow user to invite participants for project.
  • Display users notification and handle actions for them.
